# TNA-OS Canaan

Independent Home Assistant custom integration for Canaan miners running TNA-OS-CANAAN, including Avalon Nano 3s and Avalon Q.

This release tracks the TNA-OS-CANAAN 0.2.7 API and exposes additional hashrate, efficiency, fan RPM, network and pool status sensors.
